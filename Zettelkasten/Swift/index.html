# Structs vs Classes

When use each? Start by using structs and if you need to work with objetive-c or inherence, use classes

Structs are not able to inherit Similarities:

  • Blueprint
  • Group (attributes and behaviour)
  • Initialisers

Structs comes with built-in init() function and classes need to provide an init() function

Classes are passed by reference while Structs are passed by value
