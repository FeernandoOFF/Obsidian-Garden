## Obsidian controls

1. Search/Create notes (`⌘ + p`)
2. Command Palette (`⌘ +⇧ + P`)
	1. GIT
	2. Slides
	3. Admonition
3. Insert templates (`⌘ +t `)
	1. Tags
	2. draft - ([[#Digital garden]] visibility)
4. VIM-like motion
5. Sidebar L(`⌘ + ⇧ + B`) - R (`⌘ + ⇧ + X`)
6. Highlight (`⌘ + h`)
